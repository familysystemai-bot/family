📋 خطوات الإعداد الكامل:
  1. سجّل في Meta for Developers
  2. أنشئ تطبيق نوعه "Business" واربط حساب WhatsApp Business
  3. اذهب إلى Settings → Basic → انسخ App Secret والصقه أعلاه
  4. اذهب إلى WhatsApp → API Setup → انسخ Phone Number ID
  5. أنشئ System User Token دائم من Business Manager
  6. اختر أي نص عشوائي كـ Verify Token واحفظه أعلاه
  7. في إعدادات webhook على Meta:
    • Callback URL: https://your-domain.com/webhook
    • Verify Token: نفس النص اللي اخترته
  8. اضغط "Verify and Save" في Meta — يجب يرجع ✓
  9. اشترك في events: messages
🔒 ملاحظة أمنية مهمة

بدون Meta App Secret، أي شخص يعرف عنوان webhook الخاص بك يستطيع إرسال رسائل مزوّرة لنظامك. التحقق بـ HMAC-SHA256 يضمن أن الـ payload فعلاً جاء من Meta وليس من جهة أخرى.

🖼️ تحليل صور واتساب (Vision AI)

عند استلام صورة من العميل عبر واتساب، يمكن للذكاء الاصطناعي تحليلها والرد عليها. معطّل افتراضياً — فعّله فقط بعد إدخال مفتاح API صالح. عند التعطيل، تُرسَل رسالة جاهزة للعميل وتبقى الصورة في الصندوق للمعالجة اليدوية.
